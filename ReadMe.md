Install python if you don't have it : 
-----------------------------------

sudo apt-get install python

Then install Tkinter : 
---------------------

sudo apt-get install python-tk


Finally start the application with thoses commands : 
---------------------------------------------------

sudo chmod +x PyConverter.py

./PyConverter.py



Enjoy !
